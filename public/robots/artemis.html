FIRST Steamworks

FIRST Steamworks, the 2017 game for FRC, centers around a steampunk theme in which robots on each alliance must collect and deposit gears and "fuel" (wiffle balls) and climb a rope at the end of the match in order to earn points. More information about the game can be found on Wikipedia.
