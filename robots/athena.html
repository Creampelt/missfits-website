FIRST Power Up

FIRST Power Up, the 2018 game for FRC, centers around a video game theme in which robots on each alliance must deposit "power cubes" onto the "switch" and "scale" and climb a rung to "defeat the boss" in order to earn points. More information about the game can be found on Wikipedia.
